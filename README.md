# Proyecto: arielaguilar_26139_preentrega

Breve documentación para descargar, configurar y ejecutar este proyecto Java (Backend).

## Requisitos
- Java JDK 11 o superior instalado y configurado (JAVA_HOME)
- Git
- Maven o Gradle (según cómo esté configurado el proyecto)

## Clonar el repositorio

1. Abre una terminal y ejecuta:

	git clone <URL_DEL_REPOSITORIO>

2. Entra al directorio del proyecto:

	cd arielaguilar_26139_preentrega

Reemplaza `<URL_DEL_REPOSITORIO>` por la URL real del repositorio (HTTPS o SSH).
